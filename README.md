# codex-plugin-multi

Codex CLI plugin that lets Codex delegate work to **Claude Code** and **Gemini CLI**
for rescue, review, and adversarial review — the reverse of [openai/codex-plugin-cc](https://github.com/openai/codex-plugin-cc).

**Status:** design phase. See [`docs/superpowers/specs/`](./docs/superpowers/specs/) for the current design.
